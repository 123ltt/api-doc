# 批量导入

> 作者：pengyutian

**简要描述：** 

- 平台限价配置批量导入

** 自动检验：** 
- 1.校验SKU是否存在
- 2.校验平台是否合法
- 3.校验限价类型是否合法

**请求URL：** 
- `/platformLimitedPrice/batchImport`
  
**请求方式：**
- POST

|参数名|类型|说明|
