# 删除/批量删除

> 作者：lixizhou

**简要描述：** 

- 删除/批量删除平台限价配置

**请求URL：** 
- `/platformLimitedPrice/batchDelete`
  
**请求方式：**
- POST 

**请求示例：** 
```
{
    "ids": [
        1368810068543352834,
        1368810083441520642
    ]
}
```

|参数名|类型|说明|必选|
|:----    |:---|:----- |-----   |
|ids |数组[long]   |主键id|是|

 **返回示例**
``` 
{
    "code": 200,
    "success": true,
    "data": null,
    "msg": "操作成功"
}
```

|参数名|类型|说明|
|:-----  |:-----|-----|
