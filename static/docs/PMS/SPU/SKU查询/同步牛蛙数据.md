# 同步牛蛙数据

> 作者：lixizhou

**简要描述：** 

- 同步牛蛙数据

**请求URL：** 
- ` /sku/syncNwSkuOrNwSpu`
  
**请求方式：**
- POST 

**请求示例：** 
```
{
    "type": 1,
    "codeList": [
        "abc"
    ]
}
```

|参数名|类型|说明|必选|
|:----    |:---|:----- |-----   |
|type|int|1为牛蛙spu 2为牛蛙sku|是|
|codeList|数组|牛蛙spu编码或者牛蛙sku编码|是
 **返回示例**
``` 
{
    "code": 200,
    "success": true,
    "data": null,
    "msg": "操作成功"
}
```
