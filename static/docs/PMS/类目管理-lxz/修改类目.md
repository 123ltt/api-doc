# 修改类目

> 作者：fanglongbin

## 简要描述

- 修改类目

## 请求URL
- ` category/update `
  
## 请求方式
- POST 

## 参数

|参数名|必选|类型|说明|
|:----    |:---|:----- |-----   |
|id |是  |string |类目id   |
|name |是  |string | 类目名称    |

##请求事例


``` 
  {
    "id": 222222,
	"name"：家具
  }
```

## 返回示例 

``` 
  {
    "code": 200
  }
```

## 返回参数说明 

|参数名|类型|说明|
|:-----  |:-----|-----                           |
|code |int   |200 |
