# 新增类目

> 作者：yun527292121@163.com

**作者：** 李锡洲

    
**简要描述：** 

- 新增类目

**请求URL：** 
- `192.168.50.196/api/blade-product/category/save`
  
**请求方式：**
- POST 

**请求示例（JSON）**

``` 
{
	"id":"1300245618941276162",
    "name": "男士香水111"
}
```

**请求参数说明：** 

|参数名|必选|类型|说明|
|:----    |:---|:----- |-----   |
|id |number|否|上级类目id
|name|string|是|类目名称

 **返回示例**

``` 
{
    "code": 200,
    "success": true,
    "data": {
        "id": "1301369241710166017"
    },
    "msg": "操作成功"
}
```

 **返回参数说明** 

|参数名|类型|说明|
|:-----  |:-----|-----                           |
|code | number  |200成功 500失败 |
|id | number  |新增类目的id |
