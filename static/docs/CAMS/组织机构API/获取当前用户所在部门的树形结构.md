# 获取当前用户所在部门的树形结构

> 作者：wuxin

## 作者
- 吴新

    
## 简要描述

- 获取当前用户所在部门的树形结构

## 请求URL
- `/org/tree-with-filter`
  
## 请求方式
- GET 

## 参数

|参数名|必选|类型|说明|
|:----    |:---|:----- |-----   |
|parent |是  |父ID  |

## 返回参数说明 

|参数名|类型|说明|
|:-----  |:-----|-----                           |
|id |int   |主键ID  |

|title |string   |机构名称  |
|category |int   |机构类型 1集团 2公司 3部门  |
|children |array   | 子孙节点   |
