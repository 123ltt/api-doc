# (暂无)校验应用名称或者应用标识是否存在

> 作者：yangruibo

## 简要描述

- 校验应用名称或者应用标识是否存在

## 请求URL
- ` http://192.168.114.171:8179/application/verify `
  
## 请求方式
- GET 

## 参数

|参数名|必选|类型|说明|
|:----    |:---|:----- |-----   |
|appName |是  |string |应用名称   |
|code |是  |string | 应用标识    |


## 返回示例 

``` 
  {
    "code": "500",
    "success": false,
    "data": {},
    "msg": "应用名或者应用标识已经存在"
}
```

## 返回参数说明 

|参数名|类型|说明|
|:-----  |:-----|-----|
|code |int   |响应码  |
|success |int   |成功标识  |
|msg |int   |响应信息  |

## 备注 

- 更多返回错误代码请看首页的错误代码描述
