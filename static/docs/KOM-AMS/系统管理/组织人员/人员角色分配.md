# 人员角色分配

> 作者：luocheng

## 作者

- 罗成
    
## 简要描述

- 人员编辑

## 请求URL
- ` /user/assign-roles`
  
## 请求方式
- POST 

## 请求示例
```
{
  "id": "1518535756004622338",
  "roleIds": [1385141632305532929,1493900981695135746]
}
```


## 参数

|参数名|必选|类型|说明|
|:----    |:---|:----- |-----   |
|id |否  |int | 用户ID   |
|roleIds |否  |int | 关联角色ID列表  |



## 返回示例 

``` 
{
    "msg": "角色分配完成",
    "code": 200,
    "data": "",
    "success": true,
    "attachedMap": {}
}
```





## 备注 

- 更多返回错误代码请看首页的错误代码描述

|错误码|提示信息|
|:----    |:---|
|200 |用户编辑完成  |
|108009 |用户不存在  |
