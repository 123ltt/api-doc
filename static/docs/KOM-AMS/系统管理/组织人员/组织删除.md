# 组织删除

> 作者：luocheng

## 作者

- 罗成
    
## 简要描述

- 组织删除

## 请求URL
- ` /org/remove?id=1`
  
## 请求方式
- GET 

## 请求示例


## 参数

|参数名|必选|类型|说明|
|:----    |:---|:----- |-----   |
|id |是  |int | 组织ID    |



## 返回示例 

``` 
{
    "code": 200,
    "success": true,
    "data": null,
    "msg": "组织删除完成",
    "attachedMap": {}
}
```





## 备注 

- 更多返回错误代码请看首页的错误代码描述

|错误码|提示信息|
|:----    |:---|
|200 |组织删除完成  |
|108004 |组织不存在 |
|108006 |该组织下存在组织，不允许删除 |
|108007 |该组织下存在用户，不允许删除 |
