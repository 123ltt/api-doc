# 角色状态切换

> 作者：luocheng

## 作者

- 罗成
    
## 简要描述

- 角色状态切换

## 请求URL
- ` /role/state-switch`
  
## 请求方式
- POST 

## 请求示例
```
{
    "roleIds": [1518539184432934913,1518539145744674817,1518535756004622338],
    "status": 1
}
```


## 参数

|参数名|必选|类型|说明|
|:----    |:---|:----- |-----   |
|roleIds |是  |int | 角色ID列表   |
|status |是  |int |  启用状态（1 启用 2禁用）  |



## 返回示例 

``` 
{
    "msg": "角色状态切换完成",
    "code": 200,
    "data": "",
    "success": true,
    "attachedMap": {}
}
```





## 备注 

- 更多返回错误代码请看首页的错误代码描述

|错误码|提示信息|
|:----    |:---|
|200 |角色状态切换完成  |
|110009 |角色不存在  |
|2001 |业务方法参数检查不通过  |
