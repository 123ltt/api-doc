# 角色新增用户

> 作者：luocheng

## 作者

- 罗成
    
## 简要描述

- 角色新增用户

## 请求URL
- ` /role/assign-add-users`
  
## 请求方式
- POST 

## 请求示例
```
{
  "id": "1385141632305532929",
  "userIds": [17,18,26]
}
```


## 参数

|参数名|必选|类型|说明|
|:----    |:---|:----- |-----   |
|id |是  |long | 角色ID   |
|userIds |否  |string | 用户ID列表   |



## 返回示例 

``` 
{
    "msg": "用户添加完成",
    "code": 200,
    "data": "",
    "success": true,
    "attachedMap": {}
}
```
