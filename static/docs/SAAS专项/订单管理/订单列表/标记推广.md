# 标记推广

> 作者：liben

## 简要描述

- 标记推广

## 请求URL
- ` /oms/orderMain/markPromotion `
  
## 请求方式
- POST 

## 参数

|参数名|必选|类型|说明|
|:----    |:---|:----- |-----   |
|ids |是  |array(long) |ids   |
|markType |是  |int |标记类型 0 标记 1取消标记   |

## 请求示例 

``` 
{
    "ids": [
        "1524679914481225730",
        "1524679915118759938"
    ]
}
```

## 返回示例 

``` 
{
    "code": 200,
    "success": true,
    "data": true,
    "msg": "操作成功",
    "msgDetail": null,
    "attachedMap": {}
}
```

## 返回参数说明 

|参数名|类型|说明|
|:-----  |:-----|-----                           |
|groupid |int   |用户组id，1：超级管理员；2：普通用户  |
