# 转状态【废弃】

> 作者：wuxin

## 简要描述

- 修改状态

## 请求URL
- `/oms/orderMain/udpateOrderStatus`
  
## 请求方式
- POST 

## 参数

|参数名|必选|类型|说明|
|:----    |:---|:----- |-----   |
|id |是  |long |主键ID   |
|orderStatus |是  |int | 订单状态【1-待确认，2-异常订单，3-待发货，4-已发货，5-已取消】    |


## 返回示例 

``` 
  {
    "code": 0,
	"msg": null,
    "data": {
   
    }
  }
```

## 返回参数说明 

|参数名|类型|说明|
|:-----  |:-----|-----                           |
|groupid |int   |用户组id，1：超级管理员；2：普通用户  |
