# 修改子类目

> 作者：liben

## 简要描述

- 修改子类目

## 请求URL
- ` /pms/category/edit `
  
## 请求方式
- POST 

## 参数

|参数名|必选|类型|说明|
|:----    |:---|:----- |-----   |
|id |是  |long |id   |
|name |是  |string | 中文名称    |
|nameEn     |否  |string | 英文名称    |

## 返回示例 

``` 
 {
    "code": 200,
    "success": true,
    "data": null,
    "msg": "操作成功"
}
```

## 返回参数说明 

|参数名|类型|说明|
|:-----  |:-----|-----                           |
|groupid |int   |用户组id，1：超级管理员；2：普通用户  |
