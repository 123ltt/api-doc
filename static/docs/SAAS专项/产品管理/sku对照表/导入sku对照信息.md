# 导入sku对照信息

> 作者：wuxin

## 简要描述

- 导入sku对照信息

## 请求URL
- ` /pms/skuMapper/import `
  
## 请求方式
- POST 

## 参数

|参数名|必选|类型|说明|
|:----    |:---|:----- |-----   |
|file |是  |file |导入表格   |

## 返回示例 

``` 
 {
    "code": 200,
    "success": true,
    "data": null,
    "msg": "操作成功"
}
```

## 返回参数说明 

|参数名|类型|说明|
|:-----  |:-----|-----                           |
|groupid |int   |用户组id，1：超级管理员；2：普通用户  |

## 备注 

- 更多返回错误代码请看首页的错误代码描述
