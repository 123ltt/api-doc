# 删除品牌信息

> 作者：wuxin

## 简要描述

- 删除品牌信息

## 请求URL
- ` /pms/brand/delete `
  
## 请求方式
- POST 

## 参数

|参数名|必选|类型|说明|
|:----    |:---|:----- |-----   |
|ids |long  |array(long) |ids   |

## 请求示例 

``` 
[54611618,1841631,6165161]
```

## 返回示例 

``` 
 {
    "code": 200,
    "success": true,
    "data": null,
    "msg": "操作成功"，
	"msgDetail": "返回消息(详情)"
}
```

## 返回参数说明 

|参数名|类型|说明|
|:-----  |:-----|-----                           |
|groupid |int   |用户组id，1：超级管理员；2：普通用户  |
