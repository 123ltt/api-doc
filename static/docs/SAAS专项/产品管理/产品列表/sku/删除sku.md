# 删除sku

> 作者：liben

## 简要描述

- 删除sku

## 请求URL
- ` /pms/product/delete `
  
## 请求方式
- POST 

## 参数

|参数名|必选|类型|说明|
|:----    |:---|:----- |-----   |
|id |long  |array(long) |ids   |

## 请求示例 

``` 
[1513438836272873474]
```

## 返回示例 

``` 
 {
    "code": 200,
    "success": true,
    "data": null,
    "msg": "操作成功"
}
```

## 返回参数说明 

|参数名|类型|说明|
|:-----  |:-----|-----                           |
|groupid |int   |用户组id，1：超级管理员；2：普通用户  |
