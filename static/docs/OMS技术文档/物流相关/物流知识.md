体积重=长CM x 宽CM x 高CM / 6000
6000这个值和渠道关联，可以和货代谈；

拉历史订单，算出试算体积与仓库的实际体积比较，求方差；

计算试算运费时：

物流优选返回--试算运费、试算计费重、试算体积重

计算实际运费时：

物流优选返回-实际运费、实际计费重、实际体积重
