# amazon手工同步

> 作者：xiongxianxing

**简要描述：** 

- 平台订单手工同步

**请求URL：** 
- ` /oms/amplatorder/syn/1234567890`
  
**请求方式：**
- GET 

**请求示例：** 
```
http://192.168.114.172:8181/amplatorder/syn/id=1234567890
```

|参数名|类型|说明|必选|
|:----    |:---|:----- |-----   |
|id |long   |主键|是|
 **返回示例**
``` 
{
    "code": 200,
    "success": true,
    "data": true,
    "msg": "操作成功"
}
```
