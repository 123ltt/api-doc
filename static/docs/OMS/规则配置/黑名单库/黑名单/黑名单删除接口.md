# 黑名单删除接口

> 作者：fanglongbin

## 简要描述

- 黑名单删除接口

## 请求URL
- ` blacklist/remove?ids=1336604310241189889 `
  
## 请求方式
- POST 

## 参数

|参数名|必选|类型|说明|
|:----    |:---|:----- |-----   |
|ids |是  |string |主键id   |


## 返回示例 

``` 
 {

    "msg": "操作成功"
}
```

|参数名|类型|说明|
|:-----  |:-----|-----                           |
