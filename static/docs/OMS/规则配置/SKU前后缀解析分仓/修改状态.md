# 修改状态

> 作者：fanglongbin

## 简要描述

- 修改状态接口

## 请求URL
- ` /skuResolvingRule/updateStatusById `
  
## 请求方式
- POST 

## 参数
```
 {
    "id": "111111111111111",
    "status":0
}
```

## 返回示例 

``` 
{
    "code": 200,
    "success": true,
    "data": null,
    "msg": "操作成功"
}
```

## 返回参数说明 

|参数名|类型|说明|
|:-----  |:-----|-----                           |
|code |int   |200成功  |
