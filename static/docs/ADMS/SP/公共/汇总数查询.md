# 汇总数查询

> 作者：wuxin

## 简要描述

- 汇总数查询

## 请求URL
- `/am/sp/common/totalList`
  
## 请求方式
- POST 

## 参数

|参数名|必选|类型|说明|
|:----    |:---|:----- |-----   |


## 返回示例 

``` 
  {
    "code": 0,
	"msg": "操作成功",
    "data": [{
      "type": 1,
      "total": 12154545,
    }]
  }
```

## 返回参数说明 

|参数名|类型|说明|
|:-----  |:-----|-----                           |
|total |int   | 数量  |
|type |int   | 类型 1:广告活动 2:广告组 3:广告 4:关键字 5:端口投放  6:用户搜索词  |
