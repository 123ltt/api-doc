# 删除

> 作者：xuxiaofei

## 请求URL
- ` /locked/sku/remove `
  
## 请求方式
- POST 

## 参数

|参数名|必选|类型|说明|
|:----    |:---|:----- |-----   |
|ids |是  |array |主键   |

```
["1484821584733155330","1484821584745738241"]
```
