# 模板列表之状态下拉框

> 作者：1571221458@qq.com

## 简要描述

- 模板列表状态下拉框接口

## 请求URL
- ` /amazon/template/list-status `
  
## 请求方式
- GET 

## 参数

|参数名|必选|类型|说明|
|:----    |:---|:----- |-----   |

## 返回示例 

``` 
{
    "code": 200,
    "success": true,
    "data": [
        {
            "id": 0,
            "value": null,
            "name": "停用"
        },
        {
            "id": 1,
            "value": null,
            "name": "启用"
        }
    ],
    "msg": "操作成功"
}
```

## 返回参数说明 

|参数名|类型|说明|
|:-----  |:-----|-----                           |
|id |long   |隐藏值  |
|name |string   |显示值  |
