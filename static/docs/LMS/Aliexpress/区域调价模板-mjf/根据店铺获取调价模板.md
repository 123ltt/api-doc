# 根据店铺获取调价模板

> 作者：1571221458@qq.com

## 简要描述

- 获取所有的调价模板接口

## 请求URL
- ` price/adjust/getPriceAdjustTemplateByAccount `
  
## 请求方式
- GET 

## 参数

|参数名|必选|类型|说明|
|:----    |:---|:----- |-----   |
|account |是  |string |店铺名称   |

## 返回示例 

``` 
{
    "code": 200,
    "success": true,
    "data": [
        {
            "id": "1356170061227868161"
            "name": "template1"
        },
        {
            "id": "1359040097315971074",
            "name": "123"
        }
    ],
    "msg": "操作成功"
}
```

## 返回参数说明 

|参数名|类型|说明|
|:-----  |:-----|-----                           |
|id |long   |主键id  |
|name |string   |模板名称  |
