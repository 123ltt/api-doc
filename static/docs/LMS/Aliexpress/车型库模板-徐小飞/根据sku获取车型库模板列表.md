# 根据sku获取车型库模板列表

> 作者：xuxiaofei

## 简要描述

- 根据sku获取车型库模板列表接口

## 请求URL
- `/carAttr/template/get/list `
  
## 请求方式
- GET 

## 参数

|参数名|必选|类型|说明|
|:----    |:---|:----- |-----   |
|sku |否  |string |产品sku（不传返回所有车型库模板。传sku返回绑定的，未绑定的返回所有）   |

## 返回示例 

``` 
{
    "code": 200,
    "success": true,
    "data": [
        {
            "id": "1364832613769572353",
            "name": "test4"
        }
    ],
    "msg": "操作成功"
}
```

## 返回参数说明 

|参数名|类型|说明|
|:-----  |:-----|-----                           |
|id |int   |主键  |
|id |name   |模板名称  |
