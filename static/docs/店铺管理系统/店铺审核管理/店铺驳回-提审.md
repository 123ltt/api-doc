# 店铺驳回-提审

> 作者：luocheng

## 作者

- 罗成
    
## 简要描述

- 店铺复审通过/驳回

## 请求URL
- ` /store/approve/bring`
  
## 请求方式
- POST 

## 参数

|参数名|必选|类型|说明|
|:----    |:---|:----- |-----   |
|id |否  |int | 平台ID    |
|isPass |否  |Boolean | true-提审通过，false-    |


## 请求示例

``` 
{
    "id": 9,
    "isPass": true
}
```


## 返回示例 

``` 
{
    "code": 3201,
    "success": false,
    "data": {},
    "msg": "任务提审通过"
}
```
