# API授权参数删除接口

> 作者：luocheng

## 作者

- 李志远

## 请求URL
- ` /store/auth/removeParam `
  
## 请求方式
- POST 

## 参数示例

 ``` 
 {
  "id": 1314498464233373700
} 

 ```

## 参数

|参数名|必选|类型|说明|
|:----    |:---|:----- |-----   |
|id |是  |string |参数ID   |

## 返回示例 

``` 
 {
    "code": 200,
    "success": true,
    "data": {},
    "msg": "API授权参数删除成功"
}
```
