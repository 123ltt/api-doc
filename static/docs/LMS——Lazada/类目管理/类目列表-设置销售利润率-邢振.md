# 类目列表-设置销售利润率-邢振

> 作者：xingzhen

## 简要描述
- 类目列表-设置销售利润率-邢振

## 请求URL
- `/lazada/category/assign-profit-rate`
  
## 请求方式
- POST 

## 参数

|参数名|必选|类型|说明|
|:----    |:---|:----- |-----   |
|id |是  |long |主键ID   |
|profitRate |是  |double | 销售利润率    |
