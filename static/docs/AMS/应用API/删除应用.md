# 删除应用

> 作者：yangruibo

## 简要描述

- 修改应用

## 请求URL
- ` http://192.168.114.171:8179/application/remove `
  
## 请求方式
- POST 

## 参数

|参数名|必选|类型|说明|
|:----    |:---|:----- |-----   |
|ids |是  |string |应用ID组成的字符串   |

## 返回示例 

``` 
  {
    "code": 200,
    "success": true,
    "data": {},
    "msg": "操作成功"
}
```

## 返回参数说明


|参数名|类型|说明|
|:-----  |:-----|-----|
|code |int   |响应码  |
|success |int   |成功标识  |
|msg |int   |响应信息  |
