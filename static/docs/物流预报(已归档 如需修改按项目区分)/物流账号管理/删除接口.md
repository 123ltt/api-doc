# 删除接口

> 作者：1254837494@qq.com

简要描述

    删除接口

请求URL

   {tms-system}/system/platformAccount/deleted?ids=1390596076884574209(支持批量 多个id,逗号隔开)

请求方式

    get

## 参数

|参数名|必选|类型|说明|
|:-----  |:-----|-----                  |
|id |是|Long   |id

## 返回示例 

``` 
{
    "code": 200,
    "success": true,
    "data": null,
    "msg": "删除成功"
}
```
