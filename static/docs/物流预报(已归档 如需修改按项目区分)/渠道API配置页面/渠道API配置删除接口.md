# 渠道API配置删除接口

> 作者：1254837494@qq.com

简要描述

   渠道API配置删除接口

请求URL

   {tms-base}/base/channelApi/deleted?id={id}

请求方式

    GET

## 参数

|参数名|必选|类型|说明|
|:-----  |:-----|-----                  |
|id |String   |配置id|无

## 返回示例 

``` 
{
    "code": 200,
    "success": true,
    "data": null,
    "msg": "删除成功"
}
```
