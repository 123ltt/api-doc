# 菜单导出

> 作者：xiezhihua@zehui.com

**简要描述：** 

- **作者**:`谢智华`&lt;br/>菜单资源导出

**请求URL：** 
- ` {opms-service}/menu/export `
  
**请求方式：**
- get `application/x-www-form-urlencoded` 

**参数：** 

|参数名|是否必选|类型|说明|
|:----    |:---|:----- |-----   |
|menuIds| false | String | 可选菜单色集合，按照`,`分割
|isAll| false| boolean| 是否导出所有菜单 |

返回导出文件
